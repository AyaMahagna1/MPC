This is python-based implementation for the paper:https://eprint.iacr.org/2017/979.pdf
To run mpc-proj.py it requires installing python and some libraries such as hashlib, numpy, phe you can install it by this command lines:
pip install numpy
pip install hashlib
pip install phe 
you can run mpc-proj.py by the command:
python mpc-proj.py
or by programs like PyCharm.
Pay attention for installing the libraries before running the file.
** Feel free to change n and d values (number of data points and features).
In mpc-proj.py the implementation preserving privacy, in mpc-nonprivate.py it doesn't preserve privacy , you also can run mpc-nonprivate.py by the command line:
python mpc-nonprivate.py
